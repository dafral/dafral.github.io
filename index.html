About

This projects is a videogame 3D engine done by two people, Marc de Pedro and David Franco. It’s been created for educational purposes by videogame development students studying at CITM.

Here you can find a basic 3D engine, with the main functionalities done by ourselves, as well as a scripting system, also done by ourselves built on the engine of two other students. Thanks to Roger Tello and Pol Ferrando for offering us their engine. You can find their code here.
